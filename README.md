# ⚡ نظام الفواتير الإلكترونية الذكي - سمارت باور (Smart Power Invoice System)

نظام ويب تفاعلي بسيط واحترافي مصمم خصيصاً لـ **شركة سمارت باور للطاقة الشمسية والأنظمة الذكية** لتوليد، وإدارة، وطباعة فواتير المبيعات للزبائن بسهولة وسرعة.

---

## 🌟 الميزات الرئيسية (Key Features)

* **واجهة مستخدم ثنائية التصميم (Split-Screen Layout):** 
  * **جانب أيمن مظلم (Dark Mode):** مخصص لإدخال البيانات وإضافة المواد بسلاسة دون إجهاد العين.
  * **جانب أيسر فاتح (Live Preview):** يمثل شكل الفاتورة الحقيقي والنهائي الذي يظهر للزبون بشكل فوري أثناء الكتابة.
* **إدارة ديناميكية للمواد:** إضافة وحذف المواد، وتعديل الكميات بالعداد الذكي (+ / -) مع حساب فوري وتلقائي للمجموع الكلي بالدينار العراقي (د.ع).
* **إصدار أرقام مرجعية وتواريخ تلقائية:** لتسهيل التوثيق والمراجعة.
* **جاهز للطباعة الفورية (Print-Ready):** تصميم متجاوب مع أوامر الطباعة (`@media print`) يقوم بإخفاء لوحة التحكم الجانبية تلقائياً ويطبع الفاتورة فقط بكامل أناقتها وشعار الشركة في المنتصف.
* **تصدير بصيغة PDF:** يتيح حفظ الفواتير إلكترونياً ومشاركتها مع الزبائن عبر منصات التواصل.

---

## 🛠️ تقنيات التطوير (Technologies Used)

* **HTML5 & CSS3** (الهيكلية الأساسية للموقع).
* **Tailwind CSS** (لتصميم الواجهات السريعة والمتجاوبة وتنسيق الألوان الداكنة والحديثة).
* **Vanilla JavaScript** (لمعالجة العمليات الحسابية، التفاعل الحي، وتحديث البيانات في نفس اللحظة).
